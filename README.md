# Arcabouço_rpg
Um Sistema. Múltiplos Mundos. Minimalista, adaptável e focado em decisões narrativas.

O Projeto
Arcabouço é um sistema de RPG agnóstico de gênero e modular.
Ele prioriza narrativa, risco real e adaptação a diferentes cenários.
Um resgate às metodologias Old School de narrar e jogar RPG de mesa.
